# Viewport Visibility Rule

Version 13-Apr-2026

All user-facing outputs and tools (especially debugging tables, raw data outputs, and logs) MUST be verifiable as viewable in the viewport.
1.  They must use screen space efficiently.
2.  They must allow for scrolling functionality if the content overflows the available viewport dimensions.
3.  This applies regardless of the device size or window format used. Full screen interfaces are preferred for complex data.
