# Styling Policy

Version 03-May-2026

## Rule 1: Modal Close and Action Semantics

All modals must use the top-right X icon as the only close or cancel control.

Guidelines:
- Do not add Cancel or Close buttons in modal footers.
- Footer buttons are reserved for primary or secondary actions such as Save, Submit, Confirm, Continue, or Run.
- If no actionable operation exists, the modal should rely on the top-right X and Escape key behavior.
